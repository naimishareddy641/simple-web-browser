# simple-web-browser

## Description
This is a simple desktop web browser created using Electron, HTML, CSS, and JavaScript.

## Features
- URL navigation
- Back and Forward buttons
- Reload page
- Real-time web browsing

## Technologies Used
- Electron
- HTML
- CSS
- JavaScript

## How to Run
1. Install Node.js
2. Run `npm install`
3. Run `npm start`
